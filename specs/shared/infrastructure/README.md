# Shared Infrastructure

Specs for `_shared/` modules — cross-cutting infrastructure used by every feature: exceptions, middleware, JWT helpers, pagination DTOs, PDF rendering wrapper, structured logging.

When the project's `_shared/` layout is settled (after initiative 001), this folder will hold:

- `requirements.md` — what the shared infra provides and why each piece exists
- `design.md` — the canonical reference for `_shared/` modules and how they're consumed

Until initiative 001 is planned and implemented, this folder is a placeholder.
