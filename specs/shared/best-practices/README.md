# Shared Best Practices

Cross-cutting practices that apply across all apps: logging conventions, error handling, security (CSRF, auth, file validation), settings hygiene, environment variables.

When initiative 001 settles the conventions, this folder will hold:

- `requirements.md` — what practices are mandatory and why
- `design.md` — the canonical reference (specific patterns, code idioms)

Most of this content will reference (and not duplicate) `.claude/rules/django-code-architect.md` and `.claude/rules/django-test-architect.md`.
